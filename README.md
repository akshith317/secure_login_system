# Secure Login System

This project is a secure login web application built using Python Flask.

## Features

- User registration
- User login
- Password hashing using bcrypt
- SQLite database
- Input validation
- Session management
- Logout feature
- Basic SQL injection protection using Flask-SQLAlchemy

## Technologies Used

- Python
- Flask
- Flask-SQLAlchemy
- Flask-Bcrypt
- SQLite
- HTML and CSS

## How to Run

1. Install libraries:

pip install -r requirements.txt

2. Run the application:

python app.py

3. Open this in browser:

http://127.0.0.1:5000/register
